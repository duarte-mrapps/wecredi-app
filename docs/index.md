# @expo/ui Documentation

Bem-vindo à documentação oficial do `@expo/ui`. Esta biblioteca permite construir interfaces nativas de verdade, utilizando **SwiftUI** no iOS e **Jetpack Compose** no Android.

## Escolha sua Plataforma

Esta biblioteca segue os paradigmas nativos de cada plataforma. Escolha a documentação específica para o que você está construindo.

| Plataforma | Tecnologia | Documentação |
| :--- | :--- | :--- |
| **iOS** | SwiftUI | **[Guia SwiftUI (iOS)](swift-ui/index.md)** |
| **Android** | Jetpack Compose | **[Guia Jetpack Compose (Android)](jetpack-compose/index.md)** |

## Por que nativo?

- **Performance**: Renderização direta na UI Thread nativa.
- **Acessibilidade**: Suporte de primeira classe (VoiceOver/TalkBack).
- **Feel**: Animações, haptics e comportamentos que os usuários esperam.

---

> **Nota**: Embora o JavaScript controle a lógica, a renderização é 100% nativa.
